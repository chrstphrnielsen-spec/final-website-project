# Final Web Dev Project — Local files

Files added:

- index.html — professional Bootstrap-themed pages with résumé section
- scratch.html — from-scratch page (plain HTML) linking to app and professional pages
- scratch.css — stylesheet used by scratch.html
- app.html — single-file number guessing web app (inline CSS/JS)
- images/sample.svg — local image used on the scratch page

To host on GitHub Pages:

1. Create a new GitHub repository and push this folder's contents to it.
2. In the repository settings, enable GitHub Pages from the `main` branch (root).
3. After a minute, your site will be available at `https://<your-username>.github.io/<repo>/index.html`.
